
# 🌌 Yoyo's Chrome New Tab

A **modern customizable new tab dashboard** designed for productivity and quick access to tools.  
This dashboard replaces the default browser tab with a **beautiful UI, useful widgets, and quick AI access**.

It provides **time, weather, tasks, and quick AI access** in a clean minimal interface.

## ✨ Features

🕒 **Live Clock**  
Displays real-time clock on the dashboard.

🌤 **Weather Widget**  
Shows current weather information.

📅 **Date Display**  
Displays the current day and date.

📝 **To-Do List**
- Add tasks
- Track productivity
- Manage daily work

🔎 **Smart Search Bar**
Quick search options:

- Google
- ChatGPT
- YouTube
- Wikipedia

🤖 **AI Tools Panel**
Quick access to:

- Gemini
- ChatGPT
- Claude
- Perplexity

🚀 **Quick Access Shortcuts**
Fast launch for commonly used websites.

💡 **Daily Knowledge Card**
Displays random knowledge snippets.

🎨 **Modern UI Design**
- Glassmorphism
- Gradient theme
- Minimal interface

## 📦 Installation

### Install as Chrome Extension

1. Download or clone the repository

git clone https://github.com/yourusername/project-name.git

2. Open Chrome and go to

chrome://extensions/

3. Enable **Developer Mode**

4. Click **Load Unpacked**

5. Select the project folder

6. Open a new tab 🎉
